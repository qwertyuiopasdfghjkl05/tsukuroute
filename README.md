# つくルート（TSUKUROUTE）

フリーランスのデザイナー・イラストレーター・クリエイター向けの案件管理アプリです。

「相談 → 制作 → 納品 → 請求 → 入金 → 確定申告」までの仕事の道筋を、ひとつのアプリで管理できます。

## 主な機能

- 納期から逆算した制作スケジュールの自動生成（工程テンプレートは自由に編集可能）
- カレンダー（月 / 週 / 予定）でのタスク・納期管理
- 見積書・請求書の作成（PDF保存・印刷・JPG出力対応）
- 売上台帳・経費・領収書の記録と確定申告サポート
- 納品作品のギャラリー管理
- テーマカラー・案件カラーのカスタマイズ

## 特徴

- ビルド不要の HTML / CSS / JavaScript のみで動作。`index.html` を開くだけで使えます
- データはすべて利用者のブラウザ内（localStorage / IndexedDB）に保存され、サーバーには一切送信されません
- PWA対応: スマートフォンの「ホーム画面に追加」でアプリとして利用できます

## カスタマイズ

- 正式ロゴは `assets/logo.png` に配置すると、プレースホルダーの `assets/logo.svg` から自動で切り替わります
- PWAアイコンは `assets/icon-192.png` / `icon-512.png` / `apple-touch-icon.png` を差し替えられます（再生成は `python3 tools/create_pwa_icons.py`）
